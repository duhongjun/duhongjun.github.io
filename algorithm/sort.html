      # 冒泡排序

      基本思想: 每次比较两个相邻的元素,如果它们的顺序错误就把它们交换过来.

      总结一下: 如果有n个数进行排序, 只需要将n-1个数归位, 也就是说要进行n-1趟操作. 而每一趟都要从第一项开始比较相邻的两个数, 将较大的数放在后面, 比较完毕后向后挪一位继续比较下面两个相邻数的大小, 重复此步骤, 直到最后一个尚未归位的数, 已经归位的数无须比较.

      冒泡排序的核心是双循环, 时间复杂度为O(N²)

      function bubbleSort(arr) {
  // i为需要比较的趟数, 即n个数, 比较n-1趟
  for (let i = 0, len = arr.length - 1; i < arr.length; i++) {
    // j为需要比较的区间,已经归位的数不需要比较, 即0 至 (n-1-i) 这个区间
    for (let j = 0; j < len - i; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j + 1], arr[j]] = [arr[j], arr[j + 1]]
      }
    }
  }
}

      # 快速排序

      基本思想: 基于二分的思想.每次排序的时候设置一个基准点,将小于等于基准点的数全部放到基准点左边,大于等于基准点的数都放到基准点的右边.

      具体步骤如下:

      1. 将区间的最左边当做pivot
      2. 做两个标记, 一个最左left, 一个最右right
      3. 从right处开始, 找到一个比pivlot小的数
      4. 从left处开始, 找到一个比pivlot大的数
      5. 交换这两个数
      6. 继续345的步骤直到left和right相交
      7. 将pivlot和相交的数交换

      最坏时间复杂度为O(n²) time, 但是通常都是 O(n·log(n)) time (或者更快)

      function quickSort(arr, left = 0, right = arr.length - 1, bool) {
  if (left >= right) return
  const pivot = partition(arr, left, right, bool)
  // 基准归位, 开始排序基准左边和右边的数
  if(pivot !== right) {
    quickSort(arr, left, pivot - 1)
    quickSort(arr, pivot + 1, right, true)
  }
}

function partition(arr, left, right, bool) {
  // 1
  let pivot = arr[left]
  let temp = left
  while (left !== right) {
    // 3
    while (arr[right] >= pivot && left < right) {
      right--
    }
    // 4
    while (arr[left] <= pivot && left < right) {
      left++
    }
    // 5
    if(left < right) {
      [arr[left], arr[right]] = [arr[right], arr[left]]
    }
  }
  // 7
  [arr[temp], arr[left]] = [arr[left], arr[temp]]
  return left
}
